Ports del servidor

Tots els comptes de correu electrònic dels nostres servidors, independentment del nom de domini, utilitzaran els següents valors del servidor. Per utilitzar una connexió segura, utilitzeu els ajustos marcats "SSL".

Servidor entrant Nom del servidor Port
POP3 elseudomini.com 110
IMAP elseudomini.com 143
POP amb SSL elseudomini.com 995
IMAP amb SSL elseudomini.com 993
Servidor sortint Nom del servidor Port
SMTP elseudomini.com 25, 587, 8025 o 2525
SMTP amb SSL elseudomini.com 465

El servidor sortint requereix autenticació.
